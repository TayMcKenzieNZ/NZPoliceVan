# Installation ⚙️

- Download zip folder

- Remove `-main` from folder name and rename it to `NZPDVan`

- Place in your `resources` folder or in a vehicle related folder in your FiveM server

- Add `NZPDVan` to your `server.cfg`

Enjoy
